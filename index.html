Simon Mittag

I'm a veteran software engineer who learned to program in the 1980s on a C64 and have worked on many different platforms, while using new programming languages over the years. I started humbly as a mainframe system admin in the 90s, then as a Java software engineer during the first dot com boom. In the early 2000s, I worked as a developer on web applications for Dresdner Bank in Germany and later for Telstra, Allianz, News Corp, ABC and the ATO in Sydney. In the 2010's I started working on mobile apps for start-ups, spent a few years in SE Asia and have grown a system from zero to 4 million users and a team to size 30. I'm now in a management role, but always find myself returning to my software engineering roots. As of 2020, I'm mainly programming in Golang.

What do you program?

I have an interest in distributed systems, networking code and HTTP. After setting up and tuning web servers in multiple startups, I created J8a, a TLS1.3 reverse proxy and API gateway for micro-services. Three years into it's development, it has grown into a sizeable project, and a Kubernetes ingress-controller is under way.
