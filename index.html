What are you working on?

My main project as of 2023 is J8a, a TLS1.3 reverse proxy for micro-services. It's design goal is to be a fast, secure, efficient and lightweight API gateway for DevOps engineers. Three years into it's development, it has grown into a sizeable project, and a Kubernetes ingress-controller is under way.

I use a terminal on most days, and often find myself creating tools for the cli. p0d is a lab testing tool for HTTP performance with comprehensive logging. pwt blocks your CI pipeline while your servers start (so your tests don't fail too early). l0la is a sidecar for macOS, monitoring process resources during development time. All of these are available on macOS via homebrew.
